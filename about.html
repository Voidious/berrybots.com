Background

BerryBots tries to follow in the footsteps of Robocode, Robot Battle, Logo, and many other great programming games. You "play" these games by writing code to control a bot that moves around the screen. They're often used in educational settings because they're a great way to make learning to code a fun and interesting experience.

BerryBots' origins can be traced back to a discussion at the RoboWiki, a plea from a legendary Robocoder to take Robocode and package it nicely for the Raspberry Pi, or perhaps create a new, lightweight version of it. At the time, it sounded like a daunting task. But a few months later, I got Raspberry Pi fever, and a few folks tried running stock Robocode on it and saw pretty bad performance. I ordered one with the intent to start tinkering with a new programming game, with the Raspberry Pi as the initial target platform.

I began with the goal of creating a lightweight, somewhat simplified, Robocode-esque game with good performance on the Raspberry Pi. What I ended up with is a lightweight, flexible game that runs plenty fast enough on the Raspberry Pi, has much simpler APIs and game rules than Robocode, but also introduces some cool new gameplay elements, like walls, a stage that's controlled by a program much as bots are, and a secondary weapon that complements both the primary projectile weapon and the presence of walls on the stage.

Credits

  • Lua and LuaJIT: The programming language that ships and stages are written in.
  • ajstarks' OpenVG library for Raspberry Pi: The graphics library I used for the BerryBots prototype. It taught me a lot of OpenVG for the final graphics layer and I'm still using some of it, mainly the font support and init/cleanup code.
  • SFML: A cross-platform OpenGL graphics library used for the main BerryBots game graphics on other platforms (Mac, Linux, Windows).
  • wxWidgets: A cross-platform menu widgets library used for the full GUI version of BerryBots (Mac, Linux, Windows).
  • STLSoft: A few cross-platform utility functions, like high precision CPU timer and filesystem access.
  • Robocode: BerryBots is far from a Robocode clone, but I've spent the last 7 years addicted to this amazing programming game - BerryBots is certainly influenced by Robocode and would never have existed without it.
