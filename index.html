BerryBots lets you program bots in Lua and run them on Mac, Linux, Windows, and Raspberry Pi.

Every stage is different — from single-player mazes and arcade-style challenges to massive team battles. Check out the sample stages for some examples.

To get started, download BerryBots, browse the API documentation, watch the overview video, or visit the wiki to read up on BerryBots ships, stages, and gameplay concepts.
